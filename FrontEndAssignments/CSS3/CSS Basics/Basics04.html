Data is a collection of raw, unorganized facts and details like texts,observations,figures,symbols and descriptions of things etc.Data doesn't carry any specific purpose and has no significance by itself.Data is measured in bits and bytes which are basic units of information in the context of computer storage and processing .Data can be recorded and doesn't have any meaning unless procesed.
